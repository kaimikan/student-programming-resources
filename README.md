# Ресурси за летни стажанти · Враца 2026

Статичен сайт с инструменти, услуги и кратки ръководства за студентските MVP
екипи в менторската програма: AI асистенти, git крачкурс (вкл. работа от
споделени компютри), имейли и доставимост, SMS, карти, deployment, бази данни,
домейни и маркетинг.

Чист HTML + CSS, без build стъпка — `index.html` и `styles.css`.

## Локален преглед

```
python -m http.server 8000
# → http://localhost:8000
```

## Deployment

Деплойва се на Vercel като статичен проект: Import на repo-то, без framework
preset и без build команда.

Лимитите и цените в съдържанието са проверени към **юли 2026**.
